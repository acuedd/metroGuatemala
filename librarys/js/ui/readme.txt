Libreria global para mejoras a las interfaces de usuario.

Incluye funciones para:
* Interacciones:
	- Drag
	- Drop
	- Resize
	- Selectable (una lista de elementos, para ser elegidos con el mouse)
	- Sortable (Ordenar una lista de elementos)

* Widgets:
	- Accordion (como para elegir elementos en un acordion)
	- Autocomplete
	- Button
	- Dialog
	- Slider
	- Tabs
	- Datepicker
	- Progressbar

* Effectos
	- Blinds
	- Bounce
	- Clip
	- Drop
	- Explode
	- Fade
	- Fold
	- Highlight
	- Pulsate
	- Scale
	- Shake
	- Slide
	- Transfer

Utilizada por primera vez por Ludbyn en el modulo de noticias, para la galeria publica que querian en el Evelyn Rogers.
